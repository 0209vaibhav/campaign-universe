# Campaign Universe

A Burn Studio internal tool that visualises how a brand campaign extends across platforms — hero asset at the centre, satellite content orbiting outward at two orbit tiers based on their relationship to the hero.

## Stack

Next.js · TypeScript · Tailwind CSS · SVG (campaign map)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Seed Campaigns

Three campaigns are pre-loaded:

- **Jim Beam x US Soccer** — "Home Field Advantage"
- **Circle x Arc** — "Bored Room"
- **Purina Pro Plan** — "Fueled By"
