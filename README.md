# Campaign Universe

A Burn Studio internal tool that visualises how a brand campaign extends across platforms — hero asset at the centre, satellite content orbiting outward.

## Stack

Next.js · TypeScript · Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```
